# docker-helloworld

Simple hello world app

A simple nginx helloworld application to play with. Runs on port :80

To pull this image:

```bash
docker pull crackitty/docker-helloworld:latest
```

To run this image:

```bash
docker run -p 8000:80/tcp crackitty/docker-helloworld:latest
```
